# Phase 1 리서치 계획서

## 목표
손밀리 기업 통합 디자인 시스템 구축을 위한 벤치마킹 및 AI 도구 리서치

## 기간
Week 1-2 (2주)

## 리서치 범위

### 1. 디자인 시스템 벤치마킹

#### 글로벌 표준
- **Material Design 3** (Google) — 접근성, 컴포넌트 체계
- **HIG** (Apple Human Interface Guidelines) — 모바일 UX 원칙
- **Ant Design** — 엔터프라이즈 UI 패턴

#### 시니어 케어 앱 (CPO 요청 추가)
- **CareNote** (일본) — 시니어 케어 UX 패턴
- **케어링** (국내) — 한국 시니어 앱 UX
- **Silver Care** — 고령자 인터페이스 설계
- 분석 포인트: 터치 타겟 크기, 색상 대비, 폰트 크기, 네비게이션 단순화

### 2. AI 디자인 도구 리서치
- **Figma AI** — 자동 레이아웃, 컴포넌트 생성
- **v0.dev** (Vercel) — UI 코드 생성
- **Locofy** — 디자인→코드 변환
- 평가 기준: 시니어 UI 생성 품질, 접근성 자동 체크, 협업 기능

### 3. 접근성 검증 도구
- **axe** — 접근성 자동 검사
- **Lighthouse** — 성능 + 접근성 통합 검사
- **Stark (Figma 플러그인)** — 색상 대비 검사
- 목표: WCAG 2.1 AA 이상 달성

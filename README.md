# 손밀리 기업 통합 디자인 시스템

## 프로젝트 개요
- **목표**: 손밀리 기업 전체 제품의 일관된 디자인 언어 수립
- **기간**: 4주 (Phase 1)
- **적용 대상**: BOKHAUS, 출퇴근앱, 청첩장
- **주도**: CPO (UX/디자인 의사결정) + CTO (기술 구현)

## 우선순위
1. **BOKHAUS** — 시니어 UX 특수성 높음, 여기서 확립된 기준이 전체 시스템에 재사용 가능
2. **출퇴근앱** — 현장 직원 대상, 단순명료한 UX
3. **청첩장** — 모바일 최적화, 감성 디자인

## 브랜드 가이드라인 (CPO 확정)

### 색상
- 메인 컬러: 신뢰감 있는 블루/그린 계열
- 시니어 친화적 12색 팔레트 (WCAG 2.1 AA 이상 고대비)
- 배경: 흰색 또는 매우 밝은 회색 (눈의 피로 최소화)

### 타이포그래피
- 본문: 최소 16px
- 헤딩: 24px 이상
- 서체: 고딕 계열 (명조 지양, 가독성 우선)
- 행간: 1.6 이상

### 톤앤매너
- 따뜻하고 신뢰감 있는 톤
- 전문적이지만 딱딱하지 않게
- 시니어에게 친숙한 단어 선택 (전문 용어 지양)

## 성공 지표

### BOKHAUS (시니어 앱)
- 주요 기능 완료율: 85% 이상
- SUS(System Usability Scale) 점수: 75 이상
- 오류 발생률: 3% 이하

### 출퇴근앱
- 출퇴근 등록 시간: 30초 이내
- 오류율: 5% 이하
- 사용자 만족도: NPS 40 이상
